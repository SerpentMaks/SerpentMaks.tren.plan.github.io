# Training Flow

PWA для домашних тренировок без backend:
- чеклист упражнений с описанием техники;
- изменение подходов/повторов прямо в карточках;
- прогресс и график выполненных тренировок;
- локальные напоминания по дням/времени;
- экспорт/импорт JSON.

## Публикация на GitHub Pages

1. Создай репозиторий и загрузи файлы из корня проекта.
2. Включи `Settings -> Pages`.
3. Выбери `Deploy from a branch`, ветку `main`, папку `/ (root)`.
4. Дождись ссылки вида `https://<user>.github.io/<repo>`.

## Установка на iPhone

1. Открой GitHub Pages URL в Safari.
2. Нажми `Поделиться -> На экран "Домой"`.
3. Запусти приложение с экрана Домой.
4. В `Настройках` приложения выбери время и дни, затем нажми `Разрешить уведомления`.

## Важно

Версия без backend использует локальные уведомления в рамках возможностей Safari/PWA.
